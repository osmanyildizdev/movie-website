

# Movie Website

A responsive movie website built with **HTML, CSS, and JavaScript**.
This project is inspired by modern streaming platforms and focuses on creating a structured, interactive front-end interface for browsing movies.


## Overview

This project is designed as a front-end practice application to improve core web development skills.
It simulates a movie streaming interface where users can browse different movie categories, interact with UI elements, and experience a modern layout design.

The main focus of this project is not backend functionality, but rather **UI design, responsiveness, and JavaScript-based interactions**.


## Features

* Movie categories displayed in structured sections
* Horizontal scrolling movie sliders
* Hover animations on movie cards
* Dark / Light mode toggle system
* Interactive UI elements using JavaScript
* Clean and modern layout design
* Fully responsive structure for different screen sizes


## Technologies Used

* **HTML5** – page structure and semantic layout
* **CSS3** – styling, flexbox layout, transitions, responsive design
* **JavaScript (Vanilla)** – DOM manipulation and user interactions
* **Bootstrap Icons** – icon set for UI elements

##  How It Works

* Movie items are displayed inside horizontal lists
* JavaScript controls slider movement using arrow buttons
* Dark mode is implemented by toggling CSS classes
* Hover effects are handled using CSS transitions
* Layout adjusts based on screen size using media queries



##  Purpose of the Project

This project was created to strengthen front-end development skills, especially in the following areas:

* Building structured layouts with HTML
* Creating modern UI designs with CSS
* Handling user interactions with JavaScript
* Understanding responsive web design principles
* Practicing real-world front-end project structure



## Responsive Design

The project is optimized for different screen sizes.
On smaller devices, the layout adjusts by hiding or resizing certain elements to improve usability.

## Notes

* This is a static front-end project
* No backend or database integration is included
* All movie data is currently hardcoded for practice purposes
